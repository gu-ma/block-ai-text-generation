# 06 — Studio Workflow and Critical Practice

This class treats LLM use as a design/research workflow, not just “ask once, copy answer.”

## Studio workflow (recommended loop)

1. **Frame the task clearly**
   - define goal, audience, constraints, output format
2. **Generate first draft**
   - ask for 2–3 alternatives, not one
3. **Evaluate**
   - quality, relevance, originality, factual risk
4. **Refine**
   - add constraints, examples, and corrections
5. **Verify**
   - fact-check key claims and references
6. **Finalize + annotate process**
   - record where AI helped and where human edits were decisive

## Minimal evaluation rubric

Rate each output (1–5):

- Alignment with brief
- Clarity and structure
- Factual reliability
- Creativity / usefulness
- Edit effort required

Use the rubric to choose which draft to continue.

## Critical practice principles

- Do not mistake confidence for correctness.
- Keep humans accountable for final claims.
- Track sources and verification decisions.
- Be transparent about AI usage in class submissions.

## Reflection prompts

- Where did the model save you time?
- Where did it create extra correction work?
- Which prompt change most improved quality?
- What will you verify manually every time in future?
